## Building

### Dependencies

```
tclap
rudeconfig
sqlite3
```

To build, run:

```sh
git clone https://github.com/neboman11/Minecraft-Daemon
cd Minecraft-Daemon
mkdir build
cd build
cmake ..
make
```
